Genotype Files

.bgen
GEM supports version 1.1, 1.2 and 1.3 binary GEN (BGEN) genotype files described here.
In summary, the BGEN file can either be in layout 1 or layout 2 formats, contain unphased or phased genotypes, and compressed using Zlib or ZSTD.
A .sample file should accompany the .bgen file if no sample identifiers are present within the .bgen file using the --sample flag.
 
 
 
 
 

Phenotype File

.pheno
The phenotype file does not need to have the .pheno extension and can be a different type of extension (.csv, .txt, etc...) , however, the appropriate delimiter of the phenotype file should be specified using the --delim parameter. The default delimiter for phenotype files in GEM is , (comma-seperated).
A phenotype file should contain a sample identifer column, along with the phenotype, covariate, interactive covariate and exposure columns. The ordering of the columns does not matter and the first row of the phenotype file should be the header for each column.
 
 
 
 
 

Output File

.out
The --out flag should contain the full path and extension to where GEM output results. Otherwise, GEM will use gem.out as the default output file. Below is a description for each column of the output file.

Header

Description
SNPIDThe SNP identifier as retrieved from the BGEN file.
rsIDThe reference SNP ID number.
CHRThe chromosome of the SNP.
POSThe physical position of the SNP.
Allele1The first allele in the BGEN file.
Allele2The second allele in the BGEN file that is counted in association testing.
AFThe allele frequency of the second allele in the BGEN file.
Beta_MarginalThe coefficient estimate for the marginal genetic effect.
Var_Beta_MarginalThe variance associated with the marginal genetic effect estimate.
Beta_Interaction_kThe coefficient estimate for the kth exposure term,
where k = {1..length(--exposure-names)}.
Var_Beta_Interaction_k_ jThe variance associated with the kth and jth exposure term,
where j = {1..length(--exposure-names)}.
P_Value_MarginalMarginal genetic effect p-value.
P_Value_InteractionInteraction effect p-value.
P_Value_JointJoint test p-value (K+1 degrees of freedom test of genetic effect).
 
 
 
 
 

Sample File

.sample
A .sample file is required when the BGEN file does not include sample identifiers. The --sample option can be used to include a .sample file.
Formats for the .sample file should follow QCTOOLv2 format described here. The first column in the .sample file should always be the identifier column.
