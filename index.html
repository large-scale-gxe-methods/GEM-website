GEM v1.0

Gene-Environment interaction for Millions of samples


GEM (Gene-Environment interaction analysis for Millions of samples) is a software program for large-scale gene-environment interaction testing in samples from unrelated individuals. It enables genome-wide association studies in up to millions of samples while allowing for multiple exposures, control for genotype-covariate interactions, and robust inference.





Recent Updates:

Version 1.0 - March 31, 2020:

- Column name in the phenotype file that contains sample identifiers.
- Add a minor allele frequency filtering option --maf
- Multithreading is now done across SNPs by specifying the number of threads --threads
- Command-line only feature and removed the need for a parameter file
- Allow for interaction covariate adjustments --int-covar-names
- Support BGEN files with bi-allelic and phased genotypes
- Fix how GEM parses BGEN v1.1
- Support BGEN v1.3 ZSTD compression method
- Default values for --out , --robust, --tol, --delim, --missing-value, --threads, --stream-snps
  (see Parameters)
- Changed some of the output messages
- Changed the temporary bin file names to the value specified with --out
- Change header names with 'Main' to 'Marginal'
